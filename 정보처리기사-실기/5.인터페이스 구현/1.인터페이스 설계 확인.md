# 1. 인터페이스 설계 확인

- 인터페이스 설계서
  - 이 기종 시스템 및 컴포넌트 간 데이터 교환 및 처리를 위해 각 시스템의 교환되는 데이터, 업무, 송수신 주체 등이 정의된 문서
- 외부 내부 모듈 연계 방법
    - EAI(Enterprise Application Intergration) 방식
      - 포인트 투 포인트
      - 허브앤 스포크
      - 메시지 버스
      - 하이브리드
    - ESB(Enterprise Service Bus) 방식
      - 느슨한 결합
