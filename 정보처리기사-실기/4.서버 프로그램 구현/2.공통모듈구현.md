# 2. 공통 모듈 구현

* 모듈
  * 독립된 하나의 소프트웨어 또는 하드웨어 단위를 지칭하는 용어
  * 특징
    * 독립성
    * 단독 컴파일 가능 및 재사용
    * 응집도 높게
      * 모듈 내부 구성요소 간 연관 정도
      * 유형(`우논시절통순기`)
        * 우연적
        * 논리적
        * 시간적
        * 절차적
        * 통신적
        * 순차적
        * 기능적
    * 결합도 낮게
      * 모듈과의 연관도 및 모듈 간의 상호의존성
      * 유형(`내공외제스자`)
        * 내용
        * 공통
        * 외부
        * 제어
        * 스탬프
        * 자료
* MVC
  * Model
    * 애플리케이션이 무엇을 할 것인지 
    * 내부 비즈니스 로직 처리
  * View
    * 화면에 무엇을 보여줄지 것인지
    * 모델, 컨트롤러가 보여주려고 하는 것을 화면에 처리
  * Controller
    * 모델이 어떻게 처리할 것인지
    * 뷰에 명령을 보내여 화면 요청 결과 전달
* 공통 모듈 테스트
  * 화이트박스 테스트
  * 메서드 기반 테스트
  * 화면 기반 테스트
  * 테스트 드라이버 / 테스트 스텁
